# Descripción de la Aplicacion
Landing Page de una tienda e-commerce de smartphones 

Esta aplicacion esta desarrollada con React y Vite que consiste
en una Landing Page interactiva que simula el catalogo de una tienda de smartphones.

Su interfaz principal utiliza el componente ListaItems para desplegar un catálogo de smartphones mediante tarjetas individuales (ItemCard), organizadas con un sistema de paginación. 
Al hacer clic en el botón "Más información" de cualquier tarjeta, mostrara una seccion inferior (DetalleItem) con informacion detallada del smartphone seleccionado
